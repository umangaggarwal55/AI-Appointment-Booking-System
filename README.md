# AI Appointment Booking System

An AI-powered appointment booking system built with Flask and Make.com automation.

## Features

- AI-based appointment booking
- Flask backend
- Make.com workflow automation
- Webhook integration
- Secure environment variables using .env

## Tech Stack

- Python
- Flask
- HTML/CSS
- Make.com
- REST APIs
